# Typecho Web 管理平台

基于Python Flask开发的Typecho站点文章发布和Alist文件链接获取并转换成Markdown格式的集成化工具。

## 主要功能

### 1. Typecho文章管理
- **分类管理**: 支持分类树形显示、搜索、添加新分类
- **标签管理**: 支持标签多选、搜索、添加新标签
- **文章编辑**: Markdown编辑器，实时预览
- **自动匹配**: 根据文章标题自动匹配对应分类
- **一键发布**: 快速发布文章到Typecho站点

### 2. Alist文件管理
- **多账户支持**: 管理多个Alist账户
- **文件浏览**: 支持文件夹导航、文件搜索
- **链接生成**: 一键复制文件直链
- **批量操作**: 复制当前目录所有文件链接

### 3. 链接处理功能
- **一键处理链接**: 自动处理视频和图片链接，生成规范的HTML格式
- **双排模式**: 专门的双排图片布局处理
- **自动排序**: 智能识别文件名中的数字进行排序
- **统计信息**: 自动生成图片和视频数量统计

### 4. TXT文件管理
- **可视化编辑**: 直观的文本编辑界面
- **快速操作**: 支持粘贴、清除、复制操作
- **自动保存**: 实时保存，防止数据丢失
- **文件导出**: 支持自定义文件名导出

### 5. Processed页面
- **实时编辑**: 支持Markdown和HTML编辑
- **实时预览**: 所见即所得的预览效果
- **分屏显示**: 左右分屏，编辑预览同步
- **文件管理**: 支持保存和另存为功能

## 页面布局

### Typecho文章管理页面
- **左侧区域 (17%)**: 分类树和标签列表
- **编辑区域 (23%)**: 文章标题和内容编辑
- **预览区域 (30%)**: 实时预览显示
- **文件区域 (30%)**: Alist文件管理

### 其他页面
- **TXT管理**: 全屏文本编辑器
- **Processed页面**: 左右分屏编辑预览
- **响应式设计**: 适配不同屏幕尺寸

## 技术特点

1. **响应式设计**: 自适应不同分辨率和设备
2. **现代UI**: 基于Bootstrap 5的现代化界面
3. **实时交互**: AJAX异步操作，流畅的用户体验
4. **兼容性好**: 兼容主流浏览器
5. **部署简单**: 可直接部署到宝塔面板

## 安装部署

### 1. 环境要求
- Python 3.7+
- Linux服务器
- 宝塔面板 (可选)

### 2. 安装步骤

#### 直接部署
```bash
# 1. 克隆代码
git clone <repository_url>
cd typecho_web

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行应用
python app.py
```

#### 宝塔面板部署
1. 上传代码到服务器
2. 在宝塔面板中创建Python项目
3. 设置启动文件为 `app.py`
4. 配置端口 (默认6666)
5. 安装依赖包
6. 启动项目

### 3. 配置说明

#### Typecho配置
- 在"站点设置"中配置Typecho站点信息
- 配置文件: `typecho_config.json`

#### Alist配置
- 在"管理账户"中添加Alist账户信息
- 配置文件: `alist_config.json`

## 使用说明

### 1. 首次使用
1. 访问系统首页
2. 点击"Typecho文章管理"
3. 配置Typecho站点信息
4. 配置Alist账户信息

### 2. 发布文章流程
1. 选择Alist账户和目录
2. 点击"一键处理链接"按钮
3. 系统自动获取文件链接并处理
4. 编辑文章标题和内容
5. 选择分类和标签
6. 点击"发布文章"

### 3. 文件管理
1. 在TXT页面管理链接文件
2. 在Processed页面编辑处理结果
3. 支持实时预览和保存

## 注意事项

1. **权限配置**: 确保应用有文件读写权限
2. **网络连接**: 确保服务器能访问Typecho和Alist站点
3. **端口配置**: 默认端口6666，可在run.py中修改
4. **SSL证书**: 生产环境建议配置HTTPS
5. **数据备份**: 定期备份配置文件和数据文件

## 常见问题

### 1. 连接失败
- 检查Typecho和Alist站点地址是否正确
- 检查用户名和密码是否正确
- 检查网络连接是否正常

### 2. 文件上传失败
- 检查文件权限设置
- 检查磁盘空间是否充足
- 检查文件大小限制

### 3. 页面显示异常
- 清除浏览器缓存
- 检查CSS和JS文件是否加载正常
- 检查浏览器兼容性
