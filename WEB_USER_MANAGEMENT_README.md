# 网页端用户管理功能

## 🎯 功能概述

现在用户管理已经完全集成到网页端，无需使用命令行工具！管理员可以通过直观的Web界面管理系统用户。

## 🔐 访问权限

- ✅ **管理员用户** - 可以访问完整的用户管理功能
- ❌ **普通用户** - 无法访问用户管理（导航栏不会显示"用户管理"链接）

## 🌟 功能特性

### ✅ 用户管理功能
- **查看用户列表** - 显示所有用户及其角色、创建时间
- **添加新用户** - 创建新的登录账户
- **修改用户密码** - 管理员可以重置任何用户密码
- **删除用户** - 移除不需要的用户账户
- **角色管理** - 设置用户为管理员或普通用户

### ✅ 安全特性
- **权限验证** - 只有管理员可以访问用户管理
- **自我保护** - 用户不能删除自己的账户
- **密码验证** - 密码长度和格式验证
- **实时验证** - 表单输入实时验证反馈

### ✅ 用户体验
- **美观界面** - 现代化Bootstrap设计
- **响应式布局** - 适配各种设备
- **实时反馈** - 操作成功/失败即时提示
- **确认对话框** - 危险操作需要确认

## 🚀 使用方法

### 1. 访问用户管理

**作为管理员用户**：
1. 登录系统
2. 在顶部导航栏点击 "👥 用户管理"
3. 进入用户管理页面

### 2. 添加新用户

1. 点击右上角 "➕ 添加用户" 按钮
2. 填写用户信息：
   - **用户名** - 只能包含字母、数字、下划线
   - **密码** - 至少6位
   - **确认密码** - 必须与密码一致
   - **角色** - 选择"普通用户"或"管理员"
3. 点击 "添加用户" 完成创建

### 3. 修改用户密码

1. 在用户列表中找到目标用户
2. 点击 "🔑" 按钮（修改密码）
3. 输入新密码并确认
4. 点击 "修改密码" 完成

### 4. 删除用户

1. 在用户列表中找到目标用户
2. 点击 "🗑️" 按钮（删除用户）
3. 在确认对话框中点击 "确认删除"
4. ⚠️ **注意：此操作不可恢复**

## 🔧 API端点

### 获取用户列表
```
GET /api/users
权限：管理员
响应：用户列表（不包含密码）
```

### 添加用户
```
POST /api/users
权限：管理员
参数：username, password, role
```

### 删除用户
```
DELETE /api/users/<username>
权限：管理员
限制：不能删除自己
```

### 修改密码
```
PUT /api/users/<username>/password
权限：管理员或用户自己
参数：password
```

## 🛡️ 安全机制

### 权限控制
- **API级别** - 所有用户管理API都需要管理员权限
- **页面级别** - 非管理员访问会被重定向
- **导航级别** - 普通用户看不到用户管理链接

### 输入验证
- **用户名格式** - 只允许字母、数字、下划线
- **密码强度** - 最少6位字符
- **重复用户名** - 防止用户名冲突
- **自我删除保护** - 用户不能删除自己

### 操作日志

## 🎉 使用建议
